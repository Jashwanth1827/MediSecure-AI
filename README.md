# MediSecure-AI - Health Insurance Premium Prediction

AI-powered health insurance premium calculator with Indian state pricing, PDF medical report analysis, health risk prediction, and intelligent chatbot.

## Features

- Premium prediction with 32 Indian states & actuarial factors
- PDF medical report analysis (disease/severity detection)
- Health risk scoring (0-100) with prevention plans
- Gemini AI chatbot with local fallback
- What-if scenario simulator
- Fraud/anomaly detection
- Competitor comparison (10+ insurers)
- Region-based premium variation
- Explainability engine (SHAP)

## Run Locally

```bash
git clone https://github.com/Jashwanth1827/MediSecure-AI.git
cd MediSecure-AI
pip install -r requirements.txt
python app.py
```

Open http://localhost:8080

## Tech Stack

Python, Flask, Random Forest, SHAP, Google Gemini AI, SQLite, PyPDF2
